nfs: basic guide

Setting up UNIX's favorite distributed filesystem.

Client

Mounting an export

From host 192.168.1.2, mount the export /srv/nfs at /mnt/nfs/ with read-write permissions.

In /etc/fstab: 

          
            192.168.1.2:/srv/nfs	/mnt/nfs	nfs      rw    0       0

Server

Setting up an Export

Export /srv/nfs to all hosts (*) with read-write permissions.

In /etc/exports: 

          
            /srv/nfs       *(rw,sync,no_subtree_check)
Then, Refresh nfs's exports 
          
            sudo exportfs -a
